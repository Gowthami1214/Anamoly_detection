Project Overview

Unsupervised Anomaly Detection in Patient Vital Signs

This system uses advanced machine learning techniques to detect anomalies in patient vital signs without requiring labeled training data. It provides real-time analysis with explainable AI features to help healthcare professionals understand the reasoning behind each prediction.

Key Features:
  • Three Advanced Models: Isolation Forest, Autoencoder, and LSTM Autoencoder
  • Explainable AI: SHAP values and feature contribution analysis
  • Severity Categorization: Normal, Mild, Moderate, and Severe levels
  • Adaptive Thresholds: Automatic recalibration for model drift
  • Real-time Detection: Instant analysis of patient vital signs

Anomaly Detection Models

Isolation Forest

Tree-based ensemble method that isolates anomalies by randomly selecting features and split values. Uses SHAP (SHapley Additive exPlanations) for feature importance.

  • Fast training and prediction
  • Handles high-dimensional data
  • Tunable contamination parameter
Autoencoder

Neural network that learns to compress and reconstruct normal patterns. Anomalies have higher reconstruction errors. Provides per-feature error analysis.

  • Learns complex patterns
  • Per-feature explainability
  • Early stopping & dropout
LSTM Autoencoder

Sequence-based model that captures temporal patterns in vital signs. Provides time-step error heatmaps for detailed temporal analysis.

  • Captures time dependencies
  • Tunable sequence length
  • Attention mechanism

Severity Categorization

The system categorizes anomalies into four severity levels based on anomaly scores:

NORMAL (0.0 - 0.5)
All vital signs within expected ranges. No immediate concerns.
MILD (0.5 - 0.7)
Minor deviations detected. Monitor patient and consider follow-up.
MODERATE (0.7 - 0.9)
Significant anomalies detected. Medical attention recommended.
SEVERE (0.9 - 1.0)
Critical anomalies detected. Immediate medical intervention may be required.

  • Blood Pressure (Diastolic): mmHg
