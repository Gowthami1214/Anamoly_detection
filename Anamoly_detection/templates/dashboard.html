About This Dashboard

Exploratory Data Analysis (EDA)

The EDA section shows comprehensive statistical analysis and visualizations of the patient vital signs dataset, including distributions, correlations, time-series patterns, and outlier detection.

Model Performance

The model performance section displays anomaly detection results from all three models: Isolation Forest, Autoencoder, and LSTM Autoencoder. Each visualization shows anomaly scores over time with severity categorization.

Model Comparison

The comparison table shows detailed metrics for all models including F1-Score, ROC-AUC, Precision, Recall, and Silhouette Score. The best model is automatically selected and used in the prediction interface.
